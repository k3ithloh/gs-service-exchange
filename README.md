## [Tech Stack] 
Combining the best of serverless edge networks and load-balanced servers to minimize latency and maximize reliability
- Frontend:
  - Next.js + TailwindCSS
  - Deployed on Vercel

- Backend:
  - .NET
  - Deployed on AWS Elastic Beanstalk

- Database:
  - SQL database on AWS RDS
